# spring-study

vscode Community Springboot Project

## 개요
- Project SDK: java 11

- spring boot
    - version: 2.6.11

- Build Tool: Gradle
    - version: 7.2 (6.7 이상)
    - jvm: java 11 (minimum)
    - build and run using: vscode
    
- Swagger
    - http://localhost:8090/swagger-ui/index.html
    
- jar build
    - ```gradlew build -x test```

- docker
    - Dockerfile

## 구조
- controller -> service(Impl) -> dao(data access object) -> mapper -> db
